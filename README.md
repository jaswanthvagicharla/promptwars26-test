# Astra Path

The World's First Universal Bridge for Human Potential.

This is a **Vanilla Web Application** that acts as an Educational Psychologist, Financial Advisor, and Protective Guardian. It is built purely with HTML, CSS, and Javascript to be universally accessible and run instantly without any complex Node.js dependencies.

## Features Built
- **Zero-Dependency Multimodal Analysis:** Direct integration with Google Gemini 2.5 Pro REST API to process Images, Audio files, and PDFs.
- **Accessible JSON Schema Enforcement:** Guarantees that the AI translates everything into the exact structured Action Plan required by the prompt.
- **Voice-First Navigation:** Uses the `SpeechSynthesis` Web API to read out instructions so parents can listen playfully rather than read heavily.
- **Premium Dynamic Design:** High-contrast traffic lights, custom typography (Google Fonts Outfit/Inter), and micro-animations for an appealing Guardian-like aesthetic.
- **Local Sandbox Privacy:** Your API key is cached only in your local browser `localStorage`, keeping it secure.

## How to Run
Because this is a completely vanilla application, it runs flawlessly entirely in the browser.

1. **Option A (Instant):** Open the `index.html` file directly in your web browser (Chrome/Edge/Firefox).
2. **Option B (VS Code):** Use the `Live Server` extension in Visual Studio Code.

## Usage
1. Open the app.
2. The Settings Modal will ask for your **Google Gemini API Key** (Make sure to generate one from Google AI Studio).
3. Drag and drop any relevant files (e.g., photo of a messy drawing, an audio note of you or your child, a PDF report card).
4. Type any extra contextual notes into the text area.
5. Click **Discover Their Potential** and wait for the magic to happen.
6. Click **Listen to Instruction** on any of the Action Cards to hear it spoken aloud!
